# Лабораторная работа №4: Подсчёт частоты слов в тексте (WordFrequency)

## Описание проекта
Данный проект представляет собой консольное приложение на языке C++, предназначенное для синтаксического анализа текста и подсчёта частоты встречаемости каждого слова. Проект разработан в рамках выполнения лабораторной работы №4.

**Выбранный вариант:**
* **Базовая задача:** WordFrequency (Подсчёт слов в тексте)
* **Вариативная часть (Задание №4):** Выводить только слова, которые встретились больше одного раза.

## Архитектура проекта
Проект разделен на отдельные модули для обеспечения декомпозиции и чистоты структуры кода:
* `WordCounter.h` — Объявления функций для обработки текста и фильтрации данных.
* `WordCounter.cpp` — Реализация алгоритмов очистки строк, разбиения на токены и подсчета частоты.
* `tests.h` — Набор модульных тестов (`assert`) для автоматической верификации корректности бизнес-логики.
* `main.cpp` — Точка входа в программу, обеспечивающая интерактивный пользовательский ввод и вывод результатов.

## Реализованный функционал

### Обязательная часть
1.  **Потоковое чтение:** Считывание многострочного текстового ввода из стандартного потока (`std::cin`) до первой пустой строки.
2.  **Токенизация:** Разбиение полученного текста на отдельные слова при помощи `std::stringstream`.
3.  **Аналитика данных:** Подсчёт точного количества повторений каждого слова с использованием ассоциативного контейнера `std::map<std::string, int>`.
4.  **Форматированный вывод:** Отображение итогового словаря в алфавитном порядке (в формате `слово -> количество`).

### Вариативная часть (Вариант 4)
* Реализована функция фильтрации, которая отсекает уникальные слова, встречающиеся в тексте ровно один раз, и выводит в отдельном блоке только повторяющиеся лексемы.

### Дополнительно (Для упорных)
* **Очистка данных:** Автоматическое приведение слов к нижнему регистру и удаление знаков препинания во избежание дублирования (например, `Cat,`, `cat!` и `CAT` обрабатываются как одно и то же слово).
* **Тестирование:** Встроенная система Unit-тестов на базе макроса `<cassert>`. Сценарии проверяют логику базового подсчета, фильтрации и устойчивость к различным регистрам/пунктуации. Тесты автоматически запускаются при старте приложения.

## Инструкция по сборке и запуску

Для компиляции проекта вам понадобится компилятор с поддержкой стандарта C++11 или выше (например, `g++`).

### Сборка через Терминал
Перейдите в директорию с проектом и выполните команду:
```bash
g++ main.cpp WordCounter.cpp -o lab4
```

### Запуск приложения
```bash
./lab4
```
